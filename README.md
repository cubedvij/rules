## ⚔️ Правила Сервера (Мінімум)

### ✅ **Дозволено (Принципи)**
* **Дружня атмосфера:** Вхід відкритий для всіх.
* **Власність:** **Приватна власність недоторканна** (будівлі, території).
* **Спільнота:** Можна створювати держави та об'єднання.
* **Участь:** Після 20 годин гри ви маєте право голосу.

### ⛔ **Суворо Заборонено (BAN)**
* **Читерство:** Чіти, X-ray, дюп.
* **Гріферство:** Гріфінг, вбивства без причини.
* **Технічне:** Будівництво ферм/механізмів, що **шкодять продуктивності** сервера.
* **Естетика:** Будівництво **некрасивих споруд**.
* **Політика:** **Сепаратизм**. Зміна меж держав або відчуження власності **без згоди**.

***
*Адміністратор (hampta) дотримується тих самих правил.*
